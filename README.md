# AI Log Pattern Analyzer

## Overview
A **TypeScript** utility designed to automate the parsing and categorization of system logs. It helps developers quickly identify critical errors and performance warnings within large datasets.

## Key Logic
- **Pattern Categorization:** Scans log strings for specific severity markers (ERROR, WARNING, INFO).
- **Data Structuring:** Transforms raw text logs into structured objects for better visualization.
- **Reporting:** Uses `console.table` for a clear, readable audit summary.

## Why this is relevant?
Modern AI systems generate massive amounts of logs. Building tools to filter and analyze these logs is a crucial part of **Software Quality Assurance** and **System Reliability**.
